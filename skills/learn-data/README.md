# learn-data

学習カリキュラム（DD-#149）で使用するサンプルデータ。

## 注意

- このディレクトリは **スキルではない**（SKILL.md なし）
- 各 `/learn-*` スキルが Read ツールでこれらのファイルを読み込んで使用する
- データの追加・修正はこのディレクトリのファイルを編集する

## ファイル一覧

| ファイル          | 用途                                | 使用スキル        |
| ----------------- | ----------------------------------- | ----------------- |
| bad-tasks.json    | 悪いタスク記述5件 + 解説 + 模範解答 | /learn-pc-rewrite |
| medium-tasks.json | 中品質タスク記述2件 + 改善ポイント  | /learn-pc-review  |
